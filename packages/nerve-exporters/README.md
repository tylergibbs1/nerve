# @grayhaven/nerve-exporters

Every manufacturing artifact from one HIR, byte-deterministically: schematic / connector-face / harness-board SVGs (+ interactive HTML viewers), BOM / cut-list / label / test-plan CSVs and JSON twins, `graph.json`, `render-layout.json`, assembly instructions, the PDF manufacturing packet, formboard 1:1 tiling, continuity test plans, BOP, costing, machine adapters, build records, and tscircuit Circuit JSON interop (import + export).

Part of [Grayhaven Nerve](https://github.com/tylergibbs1/nerve) — harnesses as code. [Live demo + docs](https://nerve-demo.vercel.app) · [llms.txt](https://nerve-demo.vercel.app/llms.txt) · Apache-2.0
