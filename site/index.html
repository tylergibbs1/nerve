[05] Capability
[01]

Git-native hardware

Harnesses live next to firmware. Reviewable diffs, CI validation with exit codes, revision diffing down to a single pin swap.

[02]

Manufacturing packet

One command emits the PDF build book: schematic, nailboard, BOM, cut list, label schedule, assembly steps, continuity tests.

[03]

Deterministic compiler

Same input, byte-identical output — drawings, CSVs, even the zip. The renderer never owns truth; the typed graph does.

[04]

Splices & cables

Splice taps and multi-conductor cables are first-class: nets computed across them, splice-verification tests generated.

[05]

WireViz migration

Import existing WireViz YAML and instantly gain validation, cut lists, and test plans. Export back when you need to.

[06]

Open core

Apache-2.0. Seven typed packages on npm. The compiler runs locally, in CI, and in your browser.
